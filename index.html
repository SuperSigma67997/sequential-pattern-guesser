Number Sequences

You are shown a short number sequence with a hidden pattern. Your task is to identify the rule and select the correct next number. This trains abstract reasoning, pattern recognition, working memory, and tactical/logical intelligence.

Raven Progressive Matrix

Observe the 3x3 grid. The bottom-right square is missing. Identify the rule (shape progression across rows/columns) and select the correct option below. This trains pattern recognition, tactical reasoning, and adaptive intelligence.
